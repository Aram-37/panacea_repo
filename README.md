# Panacea Repository

이 레포지토리는 Panacea Cortex 시스템의 핵심 구성 요소들을 담고 있습니다.

## 주요 기능

- AI-사용자 궁합 최적화 시스템
- 의미 있는 대화 추출 및 분석
- 적응형 AI 상호작용 시스템
- IOR (Intelligent Optimization Response) 통합

## 구조

```
panacea_repo/
├── README.md
├── src/
│   ├── cortex/
│   ├── panacea/
│   └── ior/
├── data/
├── config/
└── docs/
```

## 시작하기

이 프로젝트는 AI-사용자 간의 최적화된 상호작용을 구현하기 위한 연구 및 개발 프로젝트입니다.
